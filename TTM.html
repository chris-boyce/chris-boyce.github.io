TLDR

The project aims to investigate Emmerging Technology of LLMs and how they can be used to generate dialogue with real time world data and states. Allowing for "Unlimited" Dialogue with each NPC.

The project also used OpenAIs API to generate missions that the player could complete allowing for a large range of possibilities.

User testing and statistics were collected to gain the opinion on the success of the implementation.
