TLDR

This research paper will investigate and develop a solution for implementing PCG arcade gun system in Unreal Engine, the system will be tested for the drop rate and pool as well as the usability and enjoyment from the gun system. This project will include an artifact of a third person shooter game the incorporated these game mechanics that will be used for the testing data.
