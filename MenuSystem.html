TLDR

The project aims to investigate and construct an educational framework tailored for Unreal Engine 5 using C++. It encompasses advanced programming focusing on essential programming concepts like arrays, vectors, maps, binary trees, templates, smart pointers, try-catch, and lambda functions.

Through meticulous research, each learning topic was comprehensively explored and seamlessly integrated, resulting in a well-documented process and a clear outline for the creation of a practical project within the Unreal Engine environment.

The initiative includes the development of tutorials catering to undergraduate audiences.

To ensure the reliability and correctness of the framework, thorough testing methods, including both black-box and white-box testing, will be implemented across various components, guaranteeing functionality and internal code accuracy.
