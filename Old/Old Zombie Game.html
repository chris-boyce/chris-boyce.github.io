Zombie Game

Skills Used in this Project

Teamwork, Scrum Lead, C#, Unity, Version Control

Project Overview

This Zombie Game was a group project developed over a 6 week period and required working with students on other courses. My job was Scrum Lead and Programmer.

My Contribution to this project was

  • Consistant Scrum Meetings and Documentation including UML Diagrams and Technical Design Document
  • Idea Generation and Game Design Document
  • Zombie Spawning Systems
  • Level Design and Template for Levels
  • Turret Placing Systems
  • AI Navagation and Attacks (Zombie Classes)
  • Gun Classes and Systems
  • UI Implymention including Weapon Wheel
  • Audio Controller
  • Ammo and Health Pickups
  • Animation and Ragdolls Implymention in Code
  • Supply Drops

Comments on The Project

This project took alot of time and dedication over two 3 week periods and meant alot of design decisions changed between them. This project I whole heartly loved as I finally had the knowledge and time to dedicate myself to it. I have many critics of the project but for the resources that I had the project came out really well and actually playing the game during developement became to fun and almost destracting, which can only be good signs.
