TLDR

This system will allow the user to select actors that they wish to spawn, in the engines editor. The user then will be allowed to select the amount they wish to spawn each round. The system will then wait till each one of this enemies have died and will then start the next wave.

This project was developed over a four week period. This was my first interaction with UE5, this project taught me a large amount about the engines structure as well the interactions between code and blueprint. The project could be used for tower defence games or Zombie wave gamemodes. It also allowed me to understand how to make a tool and make an editor.
